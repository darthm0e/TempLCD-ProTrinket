## ProTrinket TempLCD
Small Temperature and Humidity Display
2 Arrow Indicators showing tendency (PoC).

###### Actually parts in use:

- ProTrinket
- AdaFruit HD44780 LCD
- DHT22
- 10K Ohm Poti
- 1K Ohm Resistor
- wires
- PCBs
- usual Tools and a Brain.


###### Code used:

- AdaFruit TinyDHT Library
- Standard Arduino Stuff
